Q1: Var vs. Let vs. Const ?

Var can be declared outside or within a block and can be redefined multiple times which can cause trouble for programmers. Whereas let is block scoped which means it can be declared in a block and that variable is only accessible within the block. On the other hand const can be declared only once for a variable. It can not be redefined or updated. ES6 promotes the use of let and const.

Q2: Arrow Function vs. Regular Function ?

Regular functions are both callable and constructable where arrow functions are only callable. Arrow functions do not have their own this and do not provide arguments objects. Regular functions does provide this and arguments objects.

Q3: Map vs. forEach vs. filter vs. find ?

Map function creates a new array by returning values. forEach only works with the applied array and does not return anything. filter helps to apply certain conditions and based on that condition fulfillment it returns the desired values and find only provides the first value that meets the conditions.

Q4: Why template string is used ?

Template string is used to make dynamic and interactive texts and links.
